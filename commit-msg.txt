Navbar revamp: drop 12 inline items, ship 5 + dropdown

Old navbar had 12 items all inline on lg+, which read as
visual noise: Home, Crops, Pets, Gears, Sprinklers, Eggs,
Packs, Mutations, Calculator, Compare, Coins, Systems.
Too many visual elements competing for attention, no
clear hierarchy.

New structure: 5 primary + "More" dropdown.

PRIMARY (inline, lg+):
- Crops
- Calculator
- Mutations
- Compare
- Pets

"DOWNLOAD MORE" DROPDOWN (lg+):
- Browse group (with header): Gears, Sprinklers, Eggs, Seed Packs
- Reference group (with header): Currencies, Systems
- Each item shows icon + label + 1-line description
- 2-column grid inside a 480px popover
- Click outside or Escape to close
- Animates in with fade + slide

MOBILE (<lg):
- Hamburger menu with grouped sections
- "Browse" + "Reference" labels above the lists
- Touch-friendly 40px hit areas

VISUAL POLISH:
- Logo: gradient emerald box, "GAG" + emerald "2" + muted
  "Wiki" label, more compact
- Active nav item: emerald-400 text + gradient underline
  (inset, 1px) instead of background fill
- Hover: text-color change only (no bg fill on primary
  pills, quieter)
- Focus ring: emerald-300, 2px, accessible
- Header bottom border: emerald-400/4 shadow for a hint
  of brand color
- Dropdown: popover w/ backdrop-blur + 20px shadow +
  fade/slide-in animation
- Better spacing: gap-6 between logo and nav, gap-0.5
  between pills

ACCESSIBILITY:
- All interactive elements have focus-visible rings
- Dropdown has aria-haspopup="menu", aria-expanded
- Click-outside + Escape both close the dropdown
- aria-label on the hamburger toggle
- sr-only section headers in the dropdown

Why drop the 7 items from primary to dropdown:
- Gears, Sprinklers, Eggs, Packs, Currencies, Systems
  get less traffic than the 5 main entry points
- A "More" dropdown is the standard pattern for this
  (cf. github.com, tailwindcss.com) and reduces the
  cognitive load of the top bar
- Mobile is unchanged visually (still hamburger), but
  the mobile menu is now grouped (Browse / Reference)
  for easier scanning

Bundle: main 298.62 -> 303.16 kB (+4.5 kB for the
dropdown state, click-outside handling, animations).
Build clean, 0 TS errors.
